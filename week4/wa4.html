My choice in color and typography reflect my personal and cultural identity as in American culture certain colors could have different meanings than in other cultures. In my personal opinion I think green is more calming than blue, however, other people may think differently. Because of this my personal opinion dictates which colors I use. I think earthly colors are more authentic to me because those are the kind of colors I grew up with my entire life.

When designing my website, having accessibility easy to use is crucial to get more users. In certain color usages you have to make sure people that might not be able to see certain colors can still get the main message of the website and easily navigate the website. In addition, word font is very important as people with bad eye sight might not be able to read certain sections of your website. To add on, making sure the screen reader is able to read the website can help when choosing fonts.

What surprised me when creating a cohesive design system was getting all the different aspects to match and go well together. Trying to match colors and the style of the font is pretty difficult. Also having to get the different style for links and buttons takes a lot of code writing. Having the different hover and already visited styles takes more code to be able to get it the way I envisioned.

I think I’ll expand my system for different contexts by using semantic codes and by using the right elements in order to have my code run smoothly. I think that my coloring as well can be more flexible as they aren’t as cohesive as I want them to be. Also, I think the systems that feel the most flexible are the different styles of links you can use as well as the type of text, for example h1-h3, paragraph, etc.
